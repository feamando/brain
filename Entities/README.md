# Entities

People, Teams, and external Companies.

## File Naming
- People: `FirstName_LastName.md` (e.g., `Sameer_Doda.md`)
- Teams: `Team_Name.md` (e.g., `Team_New_Ventures.md`)
- Companies: `Company_Name.md`

## Template

```markdown
---
id: entity-firstname-lastname
type: person | team | company
role: "Role Title"
last_updated: YYYY-MM-DD
---

# Name

## Role & Responsibilities
*   Primary responsibilities

## Key Relationships
*   Reports to: [Name]
*   Collaborates with: [Names]

## Projects
*   [[Projects/Project_Name.md]]

## Notes
*   Strengths, preferences, working style

## Changelog
- **YYYY-MM-DD:** Entry
```
